# 📝 to-Do List App

This is a simple To-Do List web app made using HTML, CSS, and JavaScript.

## 🚀 Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- Clear all tasks
- Saves data in browser (localStorage)

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
